# Playwright Automation Framework

# Overview
This project is an end-to-end test automation framework built using Playwright and JavaScript. It covers key e-commerce workflows like Signup, Login, Add to Cart, and Checkout.

# Features
- Page Object Model (POM)
- Authentication reuse using storageState (auth.json)
- Dynamic test data generation
- Modular and scalable structure

# Tech Stack
- Playwright
- JavaScript (Node.js)
- GitHub

# Project Structure
tests/
WebPages/
utils/

# Run Tests
npm install
npx playwright test

# Reports
npx playwright show-report
